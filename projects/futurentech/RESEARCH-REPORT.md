## 4. YouTube Algorithm Optimization (2025)

### What the Algorithm Rewards

**Watch Time is King:**
- Videos that keep viewers watching = pushed to more people
- **Target:** 60%+ average view duration (6-8 min video = 4+ min watched)
- **Strategy:** Hook in first 15 seconds, story structure, pacing

**Click-Through Rate (CTR):**
- Thumbnail + title combo must get clicks
- **Target:** 5-10% CTR (industry standard)
- **Strategy:** Bold thumbnails, curiosity-gap titles, emotional triggers

**Engagement Signals:**
- Likes, comments, shares tell algorithm video is valuable
- **Target:** 5-10% like rate, 1-3% comment rate
- **Strategy:** End with question, pin comment asking for submissions

**Upload Frequency:**
- Channels that upload 2-3x/week grow faster than 1x/week
- **Strategy:** Batch produce, schedule releases Tuesday/Thursday/Saturday

### SEO Optimization Checklist (Per Video)

**Title:**
- Include band name + music genre + hook
- Example: "Unknown Metal Band Sounds Like Chevelle + Breaking Benjamin (AI Strategy)"
- Front-load keywords, save hook for end

**Description:**
- First 2 lines = most important (appears in search)
- Include artist links, submission form, social handles
- Use keywords naturally: "music discovery," "new band," "unsigned artist," genre terms

**Tags:**
- 10-15 tags max (YouTube de-prioritizes over-tagged videos)
- Mix broad ("new music") and specific ("unsigned metal band 2026")
- Include misspellings (common searches)

**Thumbnail:**
- High contrast, readable text on mobile
- Artist photo + bold typography
- Consistent branding (same layout every video)

**End Screen:**
- Subscribe button + next video suggestion
- 20-second end screen keeps viewers clicking
