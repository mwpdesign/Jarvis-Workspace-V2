## üí° PRO TIPS FOR LOCATION SHOOTING

### Warehouse:
- **Arrive early** - set up lights, test angles
- **Shoot wide coverage first** - get establishing shots
- **Then move to close-ups** - easier to light/frame
- **Keep instruments in tune** - check between takes
- **Sync method**: Have someone clap on camera at start of each take (makes editing easier)

### Attic:
- **Check natural light timing** - shoot during best window light
- **Use dust/atmosphere** - If attic is dusty, use it! (looks great with light)
- **Rehearse movements** - tight space, plan before shooting
- **Sound doesn't matter** - music will cover it

### Outdoors:
- **Golden hour is critical** - timing is everything
- **Scout ahead** - find best angles/backgrounds before actors arrive
- **Have backup date** - weather can ruin outdoor shoots
- **Shoot more than needed** - "magic hour" is short, maximize it

### Bottle Smash:
- **Safety first** - designated smash zone, everyone wears shoes
- **Multiple angles** - set up 2+ phones if possible
- **Slow-motion** - 120fps minimum, 240fps better
- **Practice throw** - do dry runs without bottle to nail the angle
